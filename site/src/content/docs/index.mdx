Browser Recorder is a browser extension (Chrome + Firefox) that captures
everything you need to file a bug report — console, network, interactions, DOM,
screenshots, and optional video — and bundles it into a self-contained `.zip`
you can hand to a teammate or paste into an issue.

**No server, no account, no cloud.** Everything stays on your device until you
explicitly export it.

## What you get

A single `.zip` per capture, named `browser-recording-{title}-{date}.zip`,
containing a self-contained `report.html` viewer you can open in any browser.

<CardGrid>

  <Card title="Screenshots" icon="seti:image">

  </Card>

  <Card title="Ring buffer" icon="approve-check">

## Quick start

1. Click the extension icon (or press **Alt+Shift+R**). The icon badge turns red.
2. Reproduce the bug — click around, do the things.
3. Click the icon again (or **Alt+Shift+S**). The report tab opens.
4. Edit the pre-filled reproduction notes, redact anything sensitive, and click **Export ZIP**.

For "I didn't start a session but something just went wrong" — toggle **Ring
recording** in the popup. It keeps the last 5 minutes in the background; click
**Export ring** when you need it.
