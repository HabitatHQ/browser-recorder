# Agent instructions

## Versioning

This project follows [ZeroVer](https://0ver.org/): the major version is always `0` and never advances to `1` or beyond. All releases are of the form `0.y.z`.

- Use `./scripts/bump-version.sh minor` for significant new features.
- Use `./scripts/bump-version.sh patch` for bug fixes and small improvements.
- Never bump to `1.0.0` or higher.
